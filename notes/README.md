# notes 学习笔记目录

本目录用于保存概念笔记、方法笔记、导师反馈和个人理解。笔记应帮助学生逐步建立 ChatREQ 方向的知识基础。

## 1. 笔记类型

- 概念笔记：记录需求工程、智能需求工程、知识增强、多智能体协作等基础概念。
- 方法笔记：记录需求分析、质量检查、实验设计、知识组织等方法。
- 反馈笔记：记录组会或讨论中的修改意见。
- 个人理解：用自己的话说明对方向、问题和方法的理解。

## 2. 基本要求

笔记应包含概念定义、来源、个人理解、与 ChatREQ 的关系和后续用途。不要只复制资料原文，也不要只写一句“已了解”。

## 3. 建议模板

```markdown
# 笔记标题

## 1. 概念或问题

## 2. 来源

## 3. 核心内容

## 4. 我的理解

## 5. 与 ChatREQ 的关系

## 6. 可用于哪些任务包
```
