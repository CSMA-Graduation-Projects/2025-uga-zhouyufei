# prompts 模板目录

本目录用于保存 Prompt 模板、版本记录、适用任务说明和效果评价。模板是实验材料的一部分，应可追踪、可比较、可复查。

## 1. 保存内容

- 普通需求生成模板；
- 知识增强需求生成模板；
- 智能体角色模板；
- 需求质量检查模板；
- 模板版本记录；
- 模板效果评价。

## 2. 模板记录字段

| 字段 | 说明 |
| --- | --- |
| 模板编号 | 便于实验引用 |
| 版本 | 例如 v1、v2 |
| 适用任务 | 生成、分析、检查、追问等 |
| 输入 | 模板需要的输入材料 |
| 输出 | 期望输出格式 |
| 使用知识 | 是否注入知识条目 |
| 评价结果 | 在实验中的表现 |
| 修改原因 | 为什么调整模板 |

## 3. 管理要求

不同版本模板应保留差异说明。用于实验的模板必须能与 `experiments/` 中的记录对应，避免实验结果无法复查。
