# outcomes 成果沉淀目录

本目录用于保存训练周期后期形成的成果材料。最终至少应形成论文预研报告相关材料和发明专利交底草稿。

## 1. 主要成果

- `M09-论文贡献点清单.md`
- `M09-论文实验材料清单.md`
- `M10-发明专利交底草稿.md`
- `M10-技术问题清单.md`
- `M10-技术方案流程.md`

## 2. 成果要求

成果材料应来自前期案例、知识、实验和原型，不应脱离仓库已有证据单独编写。每项成果都应能追溯到对应任务包。

## 3. 最低目标

训练周期结束前，应至少形成：

- 一份可支撑高水平会议或期刊论文的预研报告；
- 一份面向国家发明专利申请的技术交底草稿；
- 一份清楚列出实验材料、贡献点和技术方案的成果清单。
