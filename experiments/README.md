# experiments 实验记录目录

本目录用于保存 ChatREQ 相关实验记录，包括普通生成、知识增强生成、多智能体协作、质量检查和人工对比分析。

## 1. 实验记录必须包含

- 实验问题；
- 输入案例；
- 使用的知识条目；
- 智能体设置；
- Prompt 或模板版本；
- 输出结果；
- 评价维度；
- 人工分析；
- 失败分析；
- 与论文或专利目标的关系。

## 2. 评价维度

实验评价至少关注：

- 完整性；
- 一致性；
- 明确性；
- 可验证性；
- 可追踪性；
- 意图保持；
- 漏检与误检。

## 3. 实验记录模板

```markdown
# 实验标题

## 1. 实验问题

## 2. 输入案例

## 3. 实验设置

## 4. 使用知识或智能体

## 5. 输出结果

## 6. 评价结果

## 7. 人工分析

## 8. 失败原因

## 9. 对论文目标的支撑

## 10. 对专利目标的支撑
```
