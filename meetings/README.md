# meetings 组会记录目录

本目录用于保存组会记录、讨论纪要和反馈跟踪。组会记录应帮助学生明确修改任务和完成期限。

## 1. 命名建议

- `2025-XX-XX-组会记录.md`
- `2025-XX-XX-阶段讨论.md`

## 2. 记录内容

每次组会记录建议包含：

- 时间；
- 参会情况；
- 汇报内容；
- 展示材料路径；
- 导师反馈；
- 需要修改的问题；
- 修改任务；
- 完成期限；
- 下次汇报重点。

## 3. 模板

```markdown
# 组会记录

## 1. 时间

## 2. 参会情况

## 3. 本次汇报内容

## 4. 展示材料路径

## 5. 导师反馈

## 6. 修改任务

| 任务 | 负责人 | 完成期限 | 状态 |
| --- | --- | --- | --- |
|  |  |  |  |

## 7. 下次汇报重点
```
