# archive 归档目录

本目录用于保存阶段结束时的归档材料。归档的目的，是让一段时间内的周报、笔记、数据、知识条目、实验、报告、原型和成果材料能够被完整复查。

## 1. 归档时机

建议在以下时间归档：

- 每个任务包完成后；
- 每月阶段复盘后；
- 学期结束时；
- 论文预研或专利交底材料形成后；
- 毕业设计或后续项目交接前。

## 2. 归档内容

每次归档应尽量包含：

- 周报；
- 文献阅读记录；
- 学习笔记；
- 需求案例和数据；
- 知识条目；
- 智能体设计；
- Prompt 模板；
- 实验记录；
- 原型说明和截图；
- 阶段报告；
- 论文预研材料；
- 专利交底材料。

## 3. 归档说明

每个归档包应写清：

- 归档时间；
- 覆盖任务包；
- 包含文件；
- 已完成内容；
- 未完成内容；
- 后续建议。
