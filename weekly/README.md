# weekly 周报目录

本目录用于保存每周科研训练周报。周报是持续训练的重要证据，应真实记录本周围绕哪个任务包、哪个研究问题、完成了哪些具体步骤、形成了哪些文件、遇到了哪些问题。

## 1. 周报命名建议

建议使用以下命名格式：

- `2025-第01周-周报.md`
- `2025-第02周-周报.md`
- `2026-第01周-周报.md`

## 2. 周报必须回答的问题

每周周报必须回答：

1. 本周对应哪个任务包？
2. 本周围绕哪个研究问题开展工作？
3. 本周完成了哪些具体步骤？
4. 本周新增了哪些文件？
5. 本周发现了哪些失败案例或问题？
6. 本周是否进行了实验或案例分析？
7. 本周工作如何支撑论文目标？
8. 本周工作如何支撑发明专利目标？
9. 当前遇到什么问题？
10. 下周准备完成哪个明确任务？

周报不能只写“学习了相关知识”“继续完善系统”“了解了论文”“后续继续推进”等空泛内容。
